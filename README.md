# Team Player Style Assessment

A web-based assessment tool to identify your dominant team player style based on the **Parker Team Player Survey**.

## Styles Assessed
- **Contributor** — Task-oriented, reliable, organized
- **Collaborator** — Goal-focused, flexible, open to new ideas
- **Communicator** — Process-oriented, listens well, builds morale
- **Challenger** — Questions goals, candid, values integrity

## Features
- Interactive 18-question priority-ranking system
- Real-time progress tracking
- Results with Bar and Radar charts (powered by Chart.js CDN)
- Scoring normalization and style descriptions

## How to Run

### Option 1 — Open locally
```
open index.html
```
No build step or server required. Works in any modern browser.

### Option 2 — GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Source to `main` branch, root folder `/`
4. Your assessment will be live at `https://<username>.github.io/<repo-name>/`
